Commands

Use the following commands in the IRC channel or in a PM with the bot

  • join Joins the game
  • start Starts the game
  • players List who is playing the game
  • score List the scores
  • 0-9 Play that card, or if you're the Czar, pick that card as the winner
  • stop Stops the current game and resets everything
  • cards List your cards
  • help Get help
  • czar State the name of the Card Czar
